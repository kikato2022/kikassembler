##### d. 用该汇编子集编写的程序示例

求Fibonacci数列

```asm
    addi r9,r9,1        ;set r9=1
    addi r1,r1,10       ;set r1=r10, r1 is counter
loop add r10,r8,r9      ;r10 = r8+r9
    addi r8,r9,0        ;r8=r9
    addi r9,r10,0       ;r9=r10
    sw r10,r2,0         ;store r10, use r2 as offset
    addi r2,r2,1        ;offset++
    addi r1,r1,-1       ;counter--
    beqz r1,end         ;loop
    j loop
end halt
```

##### e. 书写规范

- 可选的标号，例如上例中的 loop 和 end
- 指令名称
- 指令字段：寄存器、立即数、标号
- 注释，由分号开始，到行末结束

#### 2. 该汇编小工具处理汇编文件的流程

第一遍扫描汇编文件：

1. 提取指令装入指令队列；
2. 将出现标记符号装进标记符号容器

第二遍扫描指令队列：

读取指令第一个符号

1. 若第一个符号在指令名称集里：

​		进行指令合法性检查，不合法则报错

​		合法则将该指令翻译为机器指令，输出到文件

2. 若第一个符号不在指令名称集里：

   去掉指令第一个符号，进行指令合法性检查，不合法则报错

   合法则将该指令翻译为机器指令，输出到文件
